Detailed Description

Iterates over the features/sub-features of the enabled features.

It will iterate over the features enabled. More specifically, over a FeatureSet. For each Feature or SubFeature found, it will return a iterator pointing to a std::pair.

{ "FeatureName", "SubFeatureName" }

If there is no sub-feature in the feature, an empty string is returned.
