Detailed Description

template<class KeyType, class ElemType>
class pinhao::MapVectorFeature< KeyType, ElemType >

This class maps a KeyType to a vector of ElemType features.

It is a composite feature type which maps a given KeyType to a vector of sub-features of type ElemType. So, every KeyType has all sub-feature values.
