Detailed Description

template<class T>
class pinhao::GrammarEvolution< T >

The base GrammarEvolution class.

It contains the basic information and implementations that a GrammarEvolution algorithm needs. As default, it starts when the run function is executed, and it ends with the execution of the stop function.
