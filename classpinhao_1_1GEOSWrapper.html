Detailed Description

This class is actually a wrapper to a set of functions of the GEOS framework.

The functions that calculates the costs returns a std::vector<double> with all GEOS' costs. They are:

  • RegisterUse
  • InstructionCache
  • StaticInstruction
  • TTIInstruction
  • Branch
  • Call
  • NonArchSensitive
  • ArchSensitive
