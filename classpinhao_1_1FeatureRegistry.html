Detailed Description

This class contains all registered features as well as its manipulation methods.

When a RegisterFeature variable is created it calls a static method of this class in order to insert the new feature in the registry. It is the container of all registered features.

Clones a feature with name FeatureName and returns it.

Search the RegisteredFeatures for a feature with name FeatureName. If it finds the feature, then it return the feature's clone. Else, it will return a unique_ptr to nullptr.
