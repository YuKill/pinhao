Detailed Description

template<class T, class CompareType = std::less<T>>
class pinhao::SerialSet< T, CompareType >

This class is a wrapper of the 'std::set' class, that can be serialized.

Its main members are importData and exportData. They manipulate the data inside files by calling the YAMLWrapper methods.
