FeatureType pinhao::FeatureSet::getFeature ( std::string  FeatureName,
std::string  SubfeatureName 
)
inline

Gets a feature or sub-feature which has type FeatureType.

If it should get a feature, then the variable SubfeatureName must be empty(""). Otherwise, when getting the sub-feature, it must be passed the parent FeatureName, since there may have sub-features with the same name.

Parameters
FeatureNameThe name of the feature or the parent feature.
SubfeatureNameThe name of the sub-feature if there is one.
Returns
The value of the feature/sub-feature desired.
